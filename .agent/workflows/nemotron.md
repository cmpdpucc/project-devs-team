# /nemotron - Run with Nemotron 3 Super

$ARGUMENTS

---

## Trigger
Use when the user specifically requests the use of NVIDIA Nemotron 3 Super or when high-quality coding assistance is needed for complex logic.

## Workflow

1.  **Preparation**
    -   Identify the specific coding task or question provided in $ARGUMENTS.

2.  **Execution**
    // turbo
    -   **Command**:
        ```powershell
        opencode run $ARGUMENTS --provider nv-nim --model nemotron
        ```

3.  **Completion**
    -   Display the output provided by OpenCode/Nemotron.
    -   Follow up on any code generated by the model.

---

> [!NOTE]
> This workflow assumes the `NVIDIA_API_KEY` is already set in your environment.
> If not, set it with: `$env:NVIDIA_API_KEY = "your_key"`
